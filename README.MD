# 🏨 StayEase

A modern Property Management System (PMS) built to simplify hotel, villa, apartment, and guest house operations. StayEase provides an intuitive dashboard for managing reservations, guests, rooms, payments, finance, and operational activities in one platform.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/USERNAME/StayEase.git
```

## Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend will run on:

```
http://localhost:5173
```

---

## Backend

```bash
cd backend

npm install

npm run dev
```

Backend will run on:

```
http://localhost:5000
```
